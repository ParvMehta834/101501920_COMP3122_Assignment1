##  Project Summary
This project is developed as part of **Assignment 1** for *COMP3123 – Full Stack Development*.  
It is a **RESTful API application** created using **Node.js**, **Express.js**, and **MongoDB Atlas**.

The system supports:
-  **User Registration and Login**
-  **Employee Management (CRUD)**
-  **Authentication and Input Validation**
-  **Database Connection with MongoDB Atlas**

The project follows proper RESTful design principles and returns appropriate HTTP response codes for each operation.

---

##  Technologies Implemented

| Technology | Function |
|-------------|-----------|
| **Node.js** | JavaScript runtime for server-side logic |
| **Express.js** | Framework for building web APIs |
| **MongoDB Atlas** | Cloud-based database storage |
| **Mongoose** | ODM (Object Data Modeling) library for MongoDB |
| **bcrypt.js** | Used for password encryption |
| **JSON Web Token (JWT)** | Secure token-based authentication |
| **Nodemon** | Automatically restarts the server during development |
| **Postman** | Tool for API testing and documentation |
